# 市场容量测算方法（Walmart-style）

> 适用于 deep-research 场景：**industrial-tob** 下的"终端雇主市场容量测算"独立板块。**核心思想**：用具体雇主的真实数据算市场空间，而不是引用空洞的"全球市场规模 $XB"。

## 适用场景

- 调研对象是 **To B 工业产品**（外骨骼、SaaS、企业安全设备、工业机器人等）
- 目标客户是**少数大型雇主**（如 Walmart / Amazon / 各大车企 / 各大物流公司）
- 用户希望知道"如果渗透率 X%，市场空间大概多少"

## 不适用场景

- 调研对象是 **To C 消费产品**（手机、家电、消费品）—— 这类应直接用市场报告公司的总数
- 调研对象是 **软件 / SaaS**——SaaS 应算 ARR/TAM/SAM/SOM，不是按雇主渗透率
- 数据极度稀缺的新兴市场——5% 渗透率本身就不合理

## 标杆企业选择标准

选 3-5 家**代表性雇主**，按以下标准：

1. **规模大**：员工数 10K+，避免小公司个体差异
2. **公开数据多**：员工数、岗位结构、IR 财报可查
3. **目标场景覆盖明确**：业务结构能对应到 query 涉及的场景（如仓储 → 装卸/物流）
4. **地域覆盖**：按用户指定地域，每个地域 1-2 家代表
5. **行业分散**：避免全部是同一行业（如不要全部是物流，要有制造/建筑/介护各 1）

**典型雇主池**（To B 工业产品）：
- 美国：Walmart / Amazon / UPS / FedEx / 通用汽车 / 福特 / 波音
- 欧洲：DHL Group（德国）/ Deutsche Post（德国）/ Tesco（英国）/ Carrefour（法国）
- 介护边缘：日本介护机构（Cyberdyne HAL 介护型）/ 德国 BG 体系下养老机构

## 关键变量与计算公式

```
市场空间 = 目标雇主员工总数 × 目标岗位占比 × 渗透率 × 单价
```

| 变量 | 来源 | 备注 |
|------|------|------|
| **总员工数** | 雇主 IR 页 / About / Location-Facts / 10-K | 必为 L1 实抓 |
| **目标岗位占比** | BLS / 行业报告 / 训练知识 | 标 [L1 实抓] 或 [训练知识] |
| **渗透率** | 用户 query 指定；未指定默认 5% | **5% 是 illustrative，不是预测**——必须显式说明 |
| **单价** | 厂商官网 / 训练知识 / 行业均价 | 标 [L1 实抓/训练知识]，缺时给区间 |

## 5% 渗透率选择理由

- **5% 不是预测**，是**说明性假设**——便于对话"如果 X%，空间是 Y"
- 工业设备渗透率历史曲线参考：
  - 工业机器人 2000-2020：~5% → ~30%（20 年爬坡）
  - AGV 物流机器人：~5% 阶段（2020s）
  - 安全帽/PPE：~95%（成熟期）
- **5% 是"早期采用者 → 早期大众"的过渡点**——给"未来 5 年渗透天花板"的参考

## 计算模板（Walmart 范例，来自实战）

```markdown
### 案例：Walmart 美国劳工保护外骨骼市场 [L1 实抓, 2026-06-10]

**基础数据**：
- 全球员工 **约 2.1M** [L1, Walmart IR, 2026-06-10]
- 美国员工 **约 1.6M** [L1, Walmart IR, 2026-06-10]
- 商店/俱乐部 10,800+ [L1, Walmart IR, 2026-06-10]

**目标岗位估算**：
- 美国零售业装卸工/物料搬运工占员工总数 15-20% [训练知识, BLS 行业平均]
- Walmart 美国装卸/物流岗位 ≈ **240K - 320K 人**

**5% 渗透率下市场空间**：
- 目标用户：5% × 280K = **14K 人**
- 单价：$5K - $10K [训练知识，工业级均价]
- **市场空间 ≈ $70M - $140M**（仅 Walmart 美国装卸/物流）

**为什么这个数字有用**：
- 给"如果想从 Walmart 切入，市场天花板是多少"的量化答案
- 与"全球外骨骼总市场 $0.56B [L3 实抓, MarketsandMarkets 2025-05]"对比，可看出"美国装卸物流单场景 ≈ 总市场 12-25%"——单场景足够大
- 可向投资人 / 内部决策层讲故事
```

## 分场景计算（如果 query 涉及多场景）

按 query 涉及场景拆开计算：

| 场景 | 标杆雇主 | 目标岗位 | 5% 渗透市场空间估算 |
|------|---------|---------|-------------------|
| 仓储物流 | Walmart + Amazon + UPS | 装卸工 / 物料搬运工 | $X - $Y |
| 汽车制造 | 通用 + 福特 + 比亚迪 | 装配线 / 高空作业 | $X - $Y |
| 建筑 | Bechtel / 中国铁建 | 高空作业 / 重型搬运 | $X - $Y |
| 介护（边缘） | 日本介护机构 / 德国 BG | 护理员 / 转移辅助 | $X - $Y |

**每个场景独立计算，最后求和**（如果用户要总市场）。

## 风险提示

⚠️ **5% 是 illustrative**——必须在报告中显式说明：
- 实际渗透率受采购周期（企业级 B2B 通常 6-12 个月 POC 周期）、EHS 预算、ROI 验证（工伤减少比例）、技术成熟度（外骨骼续航、舒适度）多重因素影响
- **不要把 5% 当作"5 年后达到"**——5% 是任选的中性假设，便于沟通

⚠️ **单价是区间不是定值**：
- 工业级 $5K-$10K 是粗估（基于 German Bionic / SuitX 训练知识均价）
- 实际随批量采购、订阅模式差异大
- **永远给区间，不要给单点**

⚠️ **目标岗位占比的国别差异**：
- 美国装卸工占比 vs 德国 vs 日本 vs 中国都不同
- BLS 给出的是美国数据，欧洲用 Eurostat / Destatis / ONS / INSEE
- 显式标注数据来源国家

⚠️ **数据强度与 LAW 7 配套**：
- 总员工数、目标岗位数、5% 计算结果**用 🔴/🟡/⚪ 标色**（见 SKILL.md LAW 7）
- 单价**几乎一定是 ⚪**（训练知识为主，工业产品定价不透明）
- 标杆雇主的整体数据**通常是 🔴**（IR 实抓）

## 与 deep-research 其他场景的关系

- **industry-market**：通用行业调研，市场容量测算**通常引用第三方报告**（如 MarketsandMarkets CAGR）。Walmart-style 是**自下而上**的辅助方法，两者**可互补**：
  - 第三方报告给"自上而下"的总市场（如全球 $0.56B）
  - Walmart-style 给"自下而上"的可验证分场景市场（如 Walmart $70M-$140M）
- **competitor**：竞品对比模版不做市场容量测算，**由本方法补充**
- **product-deep**：单产品深度调研，市场容量按本方法按"目标客户群"算

## 实战教训

1. **总员工数一定要实抓**——训练知识的"员工数"经常过时（公司被收购、裁员、扩张），用 2-3 年前的数字会高估/低估
2. **目标岗位占比最难拿**——BLS 行业平均不一定适用于具体公司。**标 [训练知识] + 显式说明假设**
3. **单价永远给区间**——单点会被读者当成"准确数字"反驳
4. **不要把"5% 渗透"写成"5 年后达到 5%"**——5% 是当前假设，不是未来预测
5. **Walmart 案例是范本**——任何 To B 工业产品调研都可套用此模板
