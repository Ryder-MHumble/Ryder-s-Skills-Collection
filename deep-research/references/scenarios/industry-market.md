# 场景模版：industry-market（行业市场调研）

## 适用 query 类型

- 「XX 行业格局」「XX 赛道玩家」「XX 市场分析」
- 「XX 行业融资/政策/监管」「XX 行业并购/财报」
- 关键词：行业/市场/融资/财报/监管/政策/玩家/格局/赛道/估值/营收

## 报告骨架（严格 5 章）

```
## 1. 一句话行业判断
## 2. 市场结构与玩家格局
## 3. 近期关键事件时间线（融资/并购/政策/财报）
## 4. 监管/政策环境
## 5. 趋势展望 + 关键不确定项
```

**章节数：5 个 ##，章节内不嵌套 ###**

## 章节内容指引

### 1. 一句话行业判断

- 一句话行业现状（30 字内）
- 一句话核心判断（窗口期内最重要变化）
- 窗口期
- 格式：**现状** + **核心判断** + **窗口期**

### 2. 市场结构与玩家格局

- 市场分层（如 第一梯队/第二梯队/第三梯队，或 巨头/挑战者/长尾）
- 每个层级代表玩家（市占率/估值/最近动作）
- CR3/CR5 集中度（如有数据）
- 格式示例：
  ```markdown
  **第一梯队（市占 60%+，估值 >$100B）**：
  - OpenAI - 估值 $300B（2026-04 融资 [L1, 2026-04-15]），ChatGPT 周活 5 亿 [L3, The Information 2026-04-20]，年化营收 $13B [L1, 2026-04-15]
  - Anthropic - 估值 $200B（2026-04 融资 [L1]），Claude API 增长 300% YoY [L3, 2026-04-25]

  **第二梯队（市占 10-20%，估值 $10-100B）**：
  - Google DeepMind - Gemini 系列，依托 Google 生态 [L1, 2026-04-15]
  - xAI - Grok 系列，马斯克旗下 [L1, 2026-04-15]
  - Mistral - 开源 + 商业双轨 [L1, 2026-04-15]
  ```

### 3. 近期关键事件时间线

- 按时间顺序列出窗口期内的关键事件
- 至少 8 个事件（融资/并购/产品发布/财报/政策/人事变动）
- 每个事件用日期 + 1-2 句描述 + 信源标注
- 格式：
  ```markdown
  - **2026-04-15**：[L1, Anthropic 公告] 完成 $200B 估值融资，估值 3 个月内翻倍。
  - **2026-04-18**：[L1, OpenAI 公告] ChatGPT 企业版 ARR 突破 $5B。
  - **2026-04-22**：[L1, SEC 文件] xAI 完成 $60B 融资。
  - **2026-05-03**：[L1, EU AI Office] 发布 GPAI 模型监管新规。
  - **2026-05-10**：[L3, 36氪] 国内某大模型公司完成 100 亿人民币融资。
  ```

### 4. 监管/政策环境

- 中国：网信办/工信部/各部委政策（备案、安全评估、生成式 AI 管理办法等）
- 美国：SEC/FTC/EU AI Act/EO 14110 等
- 欧盟：EU AI Act、GPAI Code of Practice
- 其他：日本/英国/韩国/印度等
- 格式：
  ```markdown
  **中国**：
  - 2024-08《生成式人工智能服务管理暂行办法》生效 [L1, 网信办 2024-08-15]
  - 2025-Q1 大模型备案累计 200+ 个 [L1, 网信办 2025-04]

  **美国**：
  - EO 14110 已被 EO 14179 撤销 [L1, 白宫 2025-01-20]
  - 2026-Q1 FTC 对 AI 厂商开出首张罚单 [L3, 2026-04]

  **欧盟**：
  - EU AI Act 2025-08 生效，分级监管 [L1, 2025-08-01]
  - GPAI Code of Practice 2026-Q1 发布 [L1, 2026-03]
  ```

### 5. 趋势展望 + 关键不确定项

- 未来 6-12 个月 3-5 个核心趋势
- 关键不确定项 3-5 个
- 格式：
  ```markdown
  **未来 6-12 个月趋势**：
  1. 头部集中加速：CR3 预计从 60% → 70% [L3, 2026-04-25]
  2. 开源追赶：Llama/Mistral/Qwen 性能逼近闭源 [L2, 2026-04]
  3. 监管收紧：EU AI Act 进入执法期 [L1, 2026-04]

  **关键不确定项**：
  - Anthropic 何时上市 [L3, 2026-04-20]
  - OpenAI 盈利时间表 [L1, 2026-04-15]
  - 中国大模型出海监管 [L1, 网信办 2026-04]
  ```

## 信源调度

### L1 必调
- 各主要公司官网公告/IR 页面/SEC 文件
- 监管机构官网（网信办、SEC、FTC、EU AI Office 等）
- 各国官方政策文件

### L2 可选
- 智库报告（ Brookings、CSIS、Carnegie、达沃斯 WEF、中国信通院、艾瑞、亿欧）
- OpenAlex 找相关行业研究

### L3 必调
- 中文：36氪/晚点/虎嗅/财新
- 英文：TechCrunch/The Information/Reuters/Bloomberg/WSJ/FT

### L4 可选
- KOL 行业评论（kol-interview-to-wiki 目录）

### L5 可选（社区舆情，谨慎使用）
- X 行业讨论（仅作为舆情参考，不作主信源）

## 强约束自检清单

- [ ] 章节数 = 5
- [ ] 玩家格局分层清晰，CR3/CR5 数据可考
- [ ] 关键事件时间线 ≥ 8 个事件，按时间排序
- [ ] 监管环境覆盖至少 3 个司法管辖区
- [ ] 趋势展望给具体可验证的预测
- [ ] 关键不确定项单独列出
- [ ] 至少 3 个信源层级（L1+L3 必到）
- [ ] 涉及中国时优先用国内信源；涉及美国/欧盟时优先用当地信源
