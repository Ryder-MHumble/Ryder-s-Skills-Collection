# v4.7 模版使用指南

## 目录结构

```
references/templates/
├── simple/                    # 简易版（米黄底+网格）
│   ├── style.css              # 共享样式
│   ├── cover.html             # 封面：ID标签 + 3行钩子 + 副标题 + 大圆头像
│   ├── content.html           # 内容卡：页眉 + 正文区 + 底部签名
│   ├── ending.html            # 结尾卡：收尾语 + 头像 + CTA + ID标签
│   └── avatar.png            # 头像（已复制到目录内）
├── complex/                   # 复杂版（信纸底+厚黑边，8元素）
│   ├── style.css
│   ├── cover.html             # 封面：8元素（见下）
│   ├── content.html           # 内容卡：黑条页眉 + 正文+引用框 + 底部签名
│   ├── ending.html            # 结尾卡：收尾语 + 头像 + CTA + 底部签名行
│   └── avatar.png
└── (旧版 cover-template.html / style.css / README.md 保留)
```

## 渲染为 PNG

```bash
# macOS Chrome headless（见 SKILL.md「Chrome headless 渲染实战」节）
CHROME="/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"
$CHROME --headless=old --disable-gpu --hide-scrollbars --no-sandbox \
  --no-first-run --disable-extensions \
  --virtual-time-budget=5000 --window-size="1200,1600" \
  --screenshot="output.png" "file:///path/to/template.html"
```

## 设计原则

1. **内容居中，装饰贴边** — brutalist 美学的核心张力
2. **8 元素上限** — 每个元素必须有明确功能，无重复无意义元素
3. **硬投影无 blur** — box-shadow 用纯色偏移（red/yellow/ink），不用模糊
4. **留白 = 呼吸感** — 标题与底部之间大量留白，不堆砌
5. **avatar.png 同目录** — HTML 用相对路径 `src="avatar.png"`，已复制到每个模版目录
