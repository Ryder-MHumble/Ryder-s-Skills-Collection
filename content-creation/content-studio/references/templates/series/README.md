# Content Studio Series Templates

Series templates are reusable Xiaohongshu visual/content systems for Ryder's recurring columns.

| Series | Status | Use When |
|---|---|---|
| AI时代的碎碎念 | existing/观点型 | social observation, labor/AI reality, mechanism explanation |
| 一分钟打假一个 GitHub 热门项目 | active/证据型 | fact-checking popular GitHub repos with tests, issues, code, benchmarks |
| Ryder的工具集推荐 | planned/工具型 | recommending tools through real usage scenarios and tradeoffs |

Current implementation: `github-factcheck/`.
