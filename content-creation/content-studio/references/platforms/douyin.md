# 抖音（Douyin）平台方法论

> **核心**：黄金 3 秒 + 价值密度 + CTA。**前 3 秒决定完播率**。
> 抖音是"刷"的状态，3 秒留人，15 秒定完播，60 秒定转化。

## 1. 平台调性速查

| 维度 | 数值 |
|------|------|
| 视频时长 | **15-60s** 短视频为主，深度内容 3min+ |
| 脚本字数 | **200-400 字**（口播 1 字 ≈ 0.3s） |
| 钩子位置 | **前 3 秒**（黄金 3 秒） |
| 封面文字 | ≤ **8 字**（大字号，竖屏可读） |
| 视觉资产 | **9:16 竖屏**（1080×1920 必选） |
| 字幕 | **全程字幕**（60% 用户静音刷） |
| 主导心理钩子 | 反认知 / 痛点共鸣 / 对比冲击 |
| 调性 | 精致、专业、节奏快、信息密度高 |

## 2. 内容结构（黄金 3 秒 + 价值密度 + CTA）

```
0-3s   ：黄金 3 秒 hook（反认知 / 痛点 / 数字 / 悬念）
3-15s  ：核心价值 1（最炸的点先说）
15-40s ：核心价值 2 + 3（递进，每 10s 一个信息点）
40-55s ：总结 / 反转 / 升华
55-60s ：CTA（关注 / 评论 / 主页）
```

**关键**：每 10s 必须有一个"信息锚点"，否则用户划走。

## 3. 4 种内容类型适配

| 内容类型 | 时长建议 | 钩子策略 | 结构要点 |
|----------|----------|----------|----------|
| **AI 产品推荐** | 30-45s | "这个 AI 工具，让我效率翻了 3 倍" | 痛点 → 产品演示 → 效果对比 → CTA |
| **GitHub 深度分析** | 45-60s | "GitHub 上这个项目，3 万星不是白来的" | 项目亮点 → 核心代码/架构 → 适用场景 → 链接引导 |
| **热点事件讲解** | 30-60s | "AI 圈今天炸了，3 分钟讲清楚" | 事件概述 → 关键信息 → 影响分析 → 你的观点 |
| **日常感悟** | 15-30s | "做了 3 年 AI 产品经理，我才明白这件事" | 反认知开头 → 个人故事 → 升华金句 → 共鸣 CTA |

## 4. 短视频口播脚本模板

```
【开头 hook】（3 秒，1 句话）
→ 反认知 / 痛点 / 数字 / 悬念，必须让人停下拇指

【要点 1】（10-15s）
→ 最核心的价值，先说结论再说原因
→ 配合画面：产品截图 / 代码片段 / 数据对比

【要点 2】（10-15s）
→ 递进或对比，制造"信息差"
→ 配合画面：before/after / 流程图 / 实操演示

【要点 3】（10-15s）
→ 升华或反转，给用户"获得感"
→ 配合画面：结果展示 / 用户反馈 / 金句卡片

【结尾 CTA】（3-5s）
→ 软性引导："关注我，下期讲 XXX"
→ 或互动引导："评论区告诉我你觉得哪个最炸"
```

### 脚本实例（AI 产品推荐）

```
[0-3s] 这个 AI 工具，让我写周报的时间从 2 小时变成 5 分钟。
[3-15s] 它叫 XXX，核心能力是自动读取你的工作记录，生成结构化周报。
        不用你手动整理，不用你回忆这周干了啥。
[15-30s] 我用了 2 周，对比之前的周报，领导说"终于看懂你在忙什么了"。
        关键它还能自动提取 OKR 进度，这个真的省命。
[30-45s] 适合所有打工人，尤其是那种"干了好多事但说不出来"的。
        免费版就够用，链接我放评论区了。
[45-50s] 关注我，下期讲怎么用 AI 自动做月报。
```

## 5. 标题 / 封面文字规范

| 元素 | 规范 |
|------|------|
| 标题 | ≤ **20 字**，含 1 个数字 + 1 个情绪词 |
| 封面文字 | ≤ **8 字**，大字号，**前 3 秒可读** |
| 字幕 | 全程字幕，**居中偏下**，避开点赞区 |
| BGM | 音量 ≤ 30%，不盖过人声 |
| 字体 | 思源黑体 / 阿里巴巴普惠体（粗体） |

**标题公式**：
- `数字 + 情绪词 + 价值`：「3 个 AI 工具，打工人必备」
- `反认知 + 群体`：「90% 的人不知道，AI 已经能做这件事了」
- `悬念 + 利益`：「学会这个 AI 技巧，少加 2 小时班」
- `热点 + 观点`：「AI 圈今天炸了，我的 3 点思考」

## 6. 反模式

- ❌ 开头自我介绍（"大家好我是 XXX"）
- ❌ 前 3 秒铺垫（用户直接划走）
- ❌ 信息密度过低（10s 没新信息 = 流失）
- ❌ 口播无字幕（60% 静音用户流失）
- ❌ 封面文字 > 8 字（竖屏看不清）
- ❌ 强推产品（"快去下载"）
- ❌ 视频时长 > 60s 但信息量不足
- ❌ BGM 盖过人声
- ❌ 结尾无 CTA
- ❌ 形容词堆砌（"超级好用的神器"）

## 8. KPI 速查

| 指标 | 健康值 | 说明 |
|------|--------|------|
| 完播率 | > **30%** | 抖音核心指标，决定是否进入下一流量池 |
| 3 秒留存 | > **50%** | hook 是否有效 |
| 点赞率 | > **3%** | 内容认可度 |
| 评论率 | > **0.5%** | 互动深度 |
| 关注转化 | > **1%** | 长期价值 |

## 9. 与其他平台差异

| 维度 | 抖音 | 小红书 | 快手 |
|------|------|--------|------|
| 调性 | 精致 / 专业 | 极简 / 共鸣 | 真实 / 接地气 |
| 时长 | 15-60s | 图文 | 15-60s |
| 钩子 | 黄金 3 秒 | 标题 + 封面 | 标题党 + 情绪 |
| 标题风格 | 克制 + 数字 | 营销号公式 | 夸张 + 炸裂 |
| 用户画像 | 一二线为主 | 一二线女性为主 | 下沉市场为主 |
