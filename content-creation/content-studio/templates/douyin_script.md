# 抖音短视频脚本模板

> 使用方法：复制此模板，填入具体内容。15-60s 口播脚本。

## 元信息

```yaml
content_type: {AI产品推荐 / GitHub分析 / 热点讲解 / 日常感悟}
hook: {7大钩子选1}
duration: {15s / 30s / 45s / 60s}
style: {录屏+口播 / 纯口播 / 口播+截图}
```

## 脚本结构（黄金3秒 + 价值密度 + CTA）

### 时间轴模板

| 时间 | 画面 | 口播 | 字幕 |
|------|------|------|------|
| 0-3s | {钩子画面：产品界面/夸张表情/关键数字} | {黄金3秒钩子句} | {≤8字大字} |
| 3-15s | {录屏/截图演示} | {核心要点1} | {关键词} |
| 15-30s | {录屏/截图演示} | {核心要点2} | {关键词} |
| 30-45s | {录屏/截图演示} | {核心要点3} | {关键词} |
| 45-60s | {出镜/产品logo} | {CTA} | {CTA文字} |

### 黄金3秒钩子句模板

```
# AI产品推荐
"用了这个AI工具，我回不去了"
"这个工具让我1小时干完了1天的活"

# GitHub分析
"这个GitHub项目今天炸了"
"开源圈又出神器了"

# 热点讲解
"刚刚发生的XX事件，影响比你想的大"
"所有人都在聊XX，但99%的人没看懂"

# 日常感悟
"用了3个月AI，我发现一个反直觉的事"
"AI时代最该自动化的不是工作，是开会"
```

### 反模式自检

- [ ] 前3秒有明确钩子（不铺垫）
- [ ] 不说"大家好"（黄金3秒不浪费）
- [ ] 口播≤60s能讲完（不超时）
- [ ] 每个要点有画面配合（不干讲）
- [ ] CTA在最后5秒（不提前）
- [ ] 字幕≤8字/屏（看不清=没有）

## 完整示例：AI产品推荐（30s）

| 时间 | 画面 | 口播 | 字幕 |
|------|------|------|------|
| 0-3s | 产品界面快速录屏 | "这个AI工具让我1小时干完了1天的活" | 1小时=1天 |
| 3-12s | 录屏：核心功能演示 | "它能自动读你的钉钉消息，分类、总结、排优先级" | 自动分类 |
| 12-22s | 录屏：对比效果 | "以前翻聊天记录找信息要20分钟，现在3秒" | 20min→3s |
| 22-30s | 出镜+产品logo | "链接在评论区，试试看" | 评论区见 |
