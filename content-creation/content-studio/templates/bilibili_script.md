# B站长视频脚本模板

> 使用方法：复制此模板，填入具体内容。5-15min 长视频分镜脚本。

## 元信息

```yaml
content_type: {AI产品推荐 / GitHub分析 / 热点讲解}
hook: {7大钩子选1}
duration: {5min / 8min / 10min / 15min}
style: {录屏+口播 / 出镜+PPT / 纯口播+画面}
```

## 脚本结构（开场白 + 逻辑展开 + 深度分析 + 互动结尾）

### 分镜模板（三栏：画面 + 口播 + 字幕/弹幕引导）

| 时间 | 画面 | 口播 | 字幕/弹幕引导 |
|------|------|------|-------------|
| 0:00-0:15 | 开场画面（项目logo/事件截图） | {开场白：1句话定位+为什么值得看} | {标题字幕} |
| 0:15-0:30 | 出镜或全屏画面 | {背景补充：这事为什么重要} | "弹幕扣1如果你也遇到过" |
| 0:30-2:00 | 录屏/PPT/截图 | {核心内容Part1：3个要点} | {要点关键词} |
| 2:00-2:15 | 过渡画面 | {小结+引导弹幕} | "弹幕告诉我你觉得哪个最炸" |
| 2:15-4:00 | 录屏/PPT/截图 | {核心内容Part2：深度拆解} | {要点关键词} |
| 4:00-4:30 | 过渡画面 | {小结+引导弹幕} | "觉得有道理扣1" |
| 4:30-5:30 | 录屏/PPT/截图 | {核心内容Part3：产品视角/个人观点} | {要点关键词} |
| 5:30-6:00 | 出镜 | {总结+个人判断} | {总结字幕} |
| 6:00-6:15 | 片尾画面 | {CTA：三连+关注} | "下期想看什么评论区告诉我" |

### 开场白模板

```
# AI产品推荐
"今天评测的这个AI工具，我用了2周，结论是：{判断}。但它不是所有人都适合，{适合人群}。"

# GitHub分析
"今天聊一个GitHub项目，{一句话定位}。它做了一件很反直觉的事：{反常识点}。"

# 热点讲解
"这两天{事件}刷屏了，但大部分报道只说了what，没说why。今天我从{视角}聊一下，这件事到底意味着什么。"
```

### 弹幕互动设计

| 时机 | 引导方式 | 目的 |
|------|---------|------|
| 开场30s | "弹幕扣1如果你也遇到过" | 制造参与感 |
| Part1结束 | "弹幕告诉我你觉得哪个最炸" | 收集反馈 |
| Part2结束 | "觉得有道理扣1" | 情绪确认 |
| 结尾 | "下期想看什么评论区告诉我" | 引导评论 |

### 反模式自检

- [ ] 开场15秒有明确钩子（不"大家好"）
- [ ] 每2分钟有弹幕互动点（不干讲）
- [ ] 口播节奏适中（不赶不拖）
- [ ] 画面有变化（不一张PPT讲到底）
- [ ] 有个人观点（不只搬运信息）
- [ ] CTA在最后15秒（不提前）

| 时间 | 画面 | 口播 | 字幕/弹幕引导 |
|------|------|------|-------------|
| 0:00-0:15 | 项目GitHub页面截图 | "今天聊一个GitHub项目，1周涨了2000星。它做了一件很反直觉的事：不用LLM就能做行为分析。" | 标题字幕 |
| 0:15-0:30 | 出镜 | "为什么这很重要？因为所有人都在往LLM上堆，这个项目反其道而行。" | "弹幕扣1如果你也关注过这个项目" |
| 0:30-2:00 | 录屏：代码结构 | "先看技术架构。核心是3个模块：数据采集、行为分类、可视化输出。最巧妙的是分类算法——" | 3个模块关键词 |
| 2:00-2:15 | 过渡 | "这部分是最有意思的，弹幕告诉我你们觉得哪个模块最难做" | 弹幕引导 |

| 4:30-6:00 | 出镜+PPT | "从产品视角看，这个东西能做成什么？我认为最有可能的方向是——" | 产品洞察字幕 |

| 6:30-6:45 | 片尾 | "下期想看什么项目分析，评论区告诉我。三连支持一下。" | "下期想看什么评论区告诉我" |
