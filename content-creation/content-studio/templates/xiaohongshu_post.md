# 小红书文案模板

> 使用方法：复制此模板，填入具体内容

## 元信息

```yaml
project: {项目名}
hook: {7 大钩子选 1}
style: {心理钩子 / 营销号 / 混合}
cta: {软性 CTA 类型}
cover: {封面文字 ≤ 12 字}
```

## 标题候选（3 选 1）

```markdown
- A（痛点共鸣）：{场景} + {情绪} + {具体行为}
- B（身份贩卖）：{高段位身份} + {暗示} + {揭晓}
- C（营销号）：{数字} + {震惊} + {价值}
```

**反模式检查**：
- [ ] 标题 ≤ 20 字
- [ ] 不带产品名
- [ ] 不带"如何" / "怎样"
- [ ] 不带"必看"（除非营销号模式）
- [ ] 有具体场景或数字

## 封面文字

```
{封面文字 ≤ 12 字}
```

## 正文（极简三段式）

### 段 1：心理钩子（1-2 句）

```
[具体场景] + [情绪动词] + [具体行为]
```

例：「打工人最大的恐惧：你救了一辈子火，老板以为你只会回'收到'」

### 段 2：对号入座清单（2-3 句）

```
- {场景 1}
- {场景 2}
- {场景 3}
- {场景 4}
```

例：
- 群里消息炸了，最后是你默默拆成清单
- 会上没说几句，会后补方案、风险、下一步
- 一边回"收到"，一边在脑子里重排优先级
- 明明只是发了几条消息，已经完成了一次小型调度

### 段 3：软性 CTA（1 句）

```
[不直接推产品] + [让用户自己行动]
```

例：
- "评论猜你是 🟠🟢🟣🔵 哪个"
- "🔥 点赞过 500 出下条对照表"

## 标签（3-5 个）

```
#大标签 #中标签 #小标签
```

例：#AI #AI产品经理 #打工人

## 反模式自检

- [ ] 没有产品名作为标题
- [ ] 没有自我介绍
- [ ] 没有"首先...其次..."
- [ ] 没有"怎么用"
- [ ] 没有强推产品
- [ ] 没有形容词（"创新的"、"智能的"）
- [ ] 没有把所有原型都罗列
- [ ] 没有 emoji 矩阵
- [ ] 没有"大家好"

## 视觉资产

- 封面：`cover.png`（3:4 = 1080×1440）
- 内页：3-5 张 `cards/card-XX.png`（3:4）
- 风格：见 `references/styles/` 选择
