# 野兽派×二次元 名片风格指南（v12）

> 这是**当前 v12**（4:5 = 720x900 profile card 比例，HTML+CSS+SVG+Chrome headless 渲染）的视觉规范。
> 取代早期 matplotlib 版（v1-v3）和按"原型图"复刻版（v4）。

## 0. 关键决策（2026-06-11 多轮迭代后定稿）

| 决策 | 选定 | 原因 |
|------|------|------|
| 比例 | **4:5 (720x900)** | 从 800x1000 缩到 720x900，避免导出空白；商业名片 1.75:1 (1050x600) 和正方形 1:1 (900x900) 都被否过 |
| 渲染技术 | **HTML+CSS+SVG + Chrome headless** | matplotlib 字体配置坑（必须 `FontProperties`）、野兽派效果粗糙、emoji 渲染不可控；Chrome headless 3s 出图，CJK 字体直接 `font-family: 'PingFang HK', ...` |
| 人物呈现 | **emoji + 几何拼贴**（不做精细插画） | v4 自画 SVG 二次元男程序员被明确否（"不要加入svg"）。v6 用 🐺 emoji 徽章 + 几何拼贴，方向被接受 |
| 真实人物图 | **4 张 3D 小人 + 特征评分选图** | `toonhub-{1,2,3,4}.png` 内嵌在 skill；根据小人视觉特征 + 用户表达/行为信号选 1 张 |
| 排版 | **非对称，紧凑，野兽派撞色** | 用户原话"排版布局要紧凑一些，文字信息密度稍微高一些，但整体必须要保持设计感不能用纯粹的对称和平铺的排版布局" |
| "新时代"调性 | **几何 + 高对比 + 留白张力** | 区别于传统野兽派的"脏乱粗野"，是"克制几何 + 撞色" |

## 1. 比例选择

```
4:5 (720x900)         ← 默认 / 小红书帖图
1:1 (900x900)         ← 备选 / 经典 profile
9:16 (720x1280)       ← 备选 / IG/TikTok 头像风格
```

❌ **不要**用：
- 1200x1200（社交头像太大、视觉空洞）
- 1050x600（商业名片比例 = "web 页面比例"，被否）
- 1200x1050（接近正方形但偏 1:1.14，模糊）

## 2. "新时代野兽派 × 二次元"风格定义

### 2.1 野兽派（Brutalism）的"新时代"变体

| 元素 | 传统野兽派 | **新时代野兽派**（本项目） |
|------|-----------|--------------------------|
| 几何 | 随机粗野 | **几何精确 + 旋转错位**（左侧新野兽派方块 + 右侧数据卡） |
| 字体 | 单一巨粗无衬线 | **字重对比 + 字距控制**（大标题 80px 配 11px 副文） |
| 颜色 | 高对比黑白 | **撞色**（米白 #F4EFE3 + 浓黑 + 警示红 + 荧光黄） |
| 排版 | 平铺直叙 | **非对称网格 + 留白张力** |
| 噪点 | 必备 | **微噪点（18% 不透明度）作纹理底**，不喧宾夺主 |

### 2.2 二次元（Celshaded Anime）的"符号化"变体

| 元素 | 传统二次元 | **符号化二次元**（本项目） |
|------|----------|--------------------------|
| 人物 | 精细插画、渐变、阴影 | **emoji + 几何形状 + 标签** |
| 表情 | 二次元大眼 + 阴影 | **几何圆 + 弧线眼 + 装饰元素**（如 v6 的动物圆形徽章） |
| 装饰 | 樱花/星星/魔法阵 | **几何拼贴 + 撞色块 + 旋转方块** |
| 真实人物 | 复杂角色立绘 | **用户提供的 3D 人物图（按性格选 1 张嵌入）** |

### 2.3 关键禁忌

- ❌ **不要精细 SVG/位图插画当人物主体**（自画糙、与野兽派调性冲突）
- ❌ **不要对称、平铺、纯网格**（无设计感）
- ❌ **不要"30s+ 极致渲染"的多层噪点叠加**（默认 <5s 出图）
- ❌ **不要"商务蓝"主色**（太规矩；野兽派要撞色）
- ❌ **不要在主视觉放赛璐璐渐变**（和"新时代"调性冲突）

## 3. 调色板（米白野兽派撞色）

| 角色 | Hex | 用法 |
|------|-----|------|
| 米白（背景主） | `#F4EFE3` | 卡纸感，比纯白更暖更"实体" |
| 浓黑（边框/文字） | `#0A0A0A` | 3px 边线 + 主标题 + emoji 描边 |
| 警示红（强调） | `#D62828` | 兽皮感、动物徽章边框、宣言印章 |
| 荧光黄（高亮） | `#F4C430` | 装饰块、动物徽章背景、标签底 |
| 纯白（卡片底） | `#FFFFFF` | 01/02/03 信息卡的底色（与米白背景对比） |
| 50% 黑（次要文字） | `#555555` | 描述、证据等次要信息 |

## 4. 关键视觉元素清单

### 4.1 背景层
- **米白底** + 18% 噪点纹理（SVG `feTurbulence`）
- **3 个旋转几何块**（右上黄方块 15°、左中红方块 -20°、中央黑方块 45°）—— 制造野兽派拼贴感

### 4.2 主体信息（必选）
- **顶部 row**：左上 `SELF-DISTILL · YYYY-MM-DD`，右上黑色 `★ MVP ★` 印章
- **姓名区**：超大字（80-110px）+ 野兽派黑体
- **左侧人物图**：占 41% 宽、59% 高，必须正常渲染；不要用大面积黑块压住画面
- **01 职场 SBTI**：左侧小信息卡 + 新野兽派方块（黄/红/绿/米色），不是整块黑底
- **02 SBTI 8 候选排行**：完整 8 行，Top 3 黑色突出，其余灰色兜底
- **03 5 维人格**：5 行 primary + secondary 同行
- **04 表达 DNA**：Top 5 词 / 消息长度 / 表达 / 情绪
- **职场宣言**：底部黑底黄字横条 + 👑 皇冠 + slogan

### 4.3 装饰元素
- 圆角矩形 + 粗黑边 + 阴影位移（3-5px）= 野兽派"贴纸"感
- 1-2° 随机旋转 = 拼贴感
- 噪点纹理底 = 印刷感

### 4.4 真实人物图
- 4 张 3D 小人必须放在 `assets/toonhub-{1,2,3,4}.png`
- 选图由 `select_figure()` 完成：先看小人视觉特征，再用 `expression_dna` / `behavior_patterns` / SBTI Top1 加成评分
- 橙色眼镜 `toonhub-1`：专注深潜，适合长消息、工作词、抽象词、深度分析
- 绿色比 V `toonhub-2`：元气讲解，适合积极情绪、讲解、表达、氛围带动
- 粉色举拳 `toonhub-3`：项目推进，适合稳定白天、核心协作、指挥官/记者型推进
- 蓝色指向 `toonhub-4`：侦察响应，适合问句多、短回复、群聊、深夜快速响应

## 6. 渲染实现（HTML+CSS+SVG+Chrome headless）

### 6.1 模板结构
- 单文件 HTML（CSS 内联在 `<style>`，SVG 内联在 body）
- 写入临时文件 → Chrome headless 截图 → 删除临时文件
- 命令：
  ```bash
  "/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
    --headless=new --disable-gpu --no-sandbox --hide-scrollbars \
    --force-device-scale-factor=1 \
    --window-size=800,1000 \
    --screenshot=/tmp/output.png \
    file:///tmp/input.html
  ```

### 6.2 字体
- 头 style 里直接指定 `font-family: 'PingFang HK', 'Hiragino Sans GB', 'STHeiti', 'Noto Sans CJK SC', sans-serif;`
- macOS 自带 PingFang HK 等，无需额外安装
- emoji 用系统原生（macOS 自带 Apple Color Emoji）

### 6.3 噪点纹理
- 内联 SVG `feTurbulence` 滤镜，背景层 18% 不透明度：
  ```css
  background-image: url("data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='200' height='200'><filter id='n'><feTurbulence type='fractalNoise' baseFrequency='0.9'/></filter><rect width='200' height='200' filter='url(%23n)' opacity='0.5'/></svg>");
  ```

### 6.4 Python format 字符串陷阱 ⚠️
- CSS 模板用 `str.format()` 时所有 `{}` 必须转义成 `{{}}`
- 否则报 `ValueError: Single '}' encountered in format string`
- 占位符用 `{name}` 形式（单花括号）但要避开 CSS 语法的 `{`

### 6.5 性能
- 单张 720x900 输出约 3s（Chrome headless 主开销）
- 不做多层叠加（噪点/扫描线/磨砂）—— 用户已经接受这个速度

## 8. 失败 / 已知限制

- vision API 经常静默（self-distill 这次也遇到），无法自动验证视觉是否符合预期
- 4 张 toonhub 图已按视觉特征建模；如果替换图片，必须重新校准 `FIGURE_LIBRARY` 和测试。
- Chrome headless 在 macOS 上有 1-3s 启动开销，3 张图跑完 9-12s
- 大尺寸图（如 4K toonhub 原图 5-7MB）作为 base64 内嵌会让 HTML 文件变大但不影响 Chrome 截图性能
