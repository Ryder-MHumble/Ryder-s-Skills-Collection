# 分析方法论

> 自我分析 ≠ 心理诊断。本文档定义"自查" Skill 内置的分析框架，每条结论必须能追溯到具体证据。

## 标签语义

每条结论前用以下标签区分可信度：

| 标签 | 含义 | 例 |
|------|------|-----|
| 🔴 **硬数据** | 来自 dws API 的客观统计，可独立验证 | "本月共发 1247 条消息" |
| 🟡 **软分析** | 基于词频/句式的统计推断，可能误判但有数据支撑 | "你最常用句末词是'啊'" |
| 🔮 **趣味推断** | 跨域类比（MBTI/动物等），信效度有限但好玩 | "你像一只独居的猫头鹰" |
| ⚠️ **数据不全** | 该维度数据采集失败或样本不足 | "本周期听记数据未采集" |

**用户面前展示时，每个推断必须带标签**——硬数据可以直接信，软分析要解释证据，趣味推断要明确「这不一定准」。

## 一、表达 DNA 分析

### 1.1 高频词（n-gram）

**方法**：
- 文本分词（结巴 jieba 优先，fallback 简单按字符切分）
- 过滤停用词（"了/的/是/在/和/就"等 200+ 通用停用词）
- 计算 1-gram / 2-gram / 3-gram 频次
- 取 Top 50

**输出维度**：
- 工作关键词（"周报/项目/会议/进度/对接/同步"等业务词）
- 情感词（"好/赞/同意/没问题" vs "不行/有疑问/不行/卡"）
- 口头禅（连续 3 个字+重复 ≥3 次 的短语，如 "就是说""对吧""怎么说"）
- 标点偏好（！？… 密度）

**判断标准**：
- 某个词占总消息数 >30% → "口头禅"
- 某个词出现 ≥5 次且和上下文无关 → "习惯性表达"
- 业务词 >30% → "工作导向型表达"
- 情感词稀少 → "客观中立型"

### 1.2 句式分析

**特征**：
- 平均消息长度（中位数/均值/P90）
- 问句频率（以"？"或"吗"结尾）
- 感叹句频率（"！"密度）
- 长消息比例（>50 字符）
- 单字回复比例（"好/收/嗯/OK"）

**判断标准**：
- 均长 <10 字符 + 单字回复 >40% → "极简风格"
- 均长 >50 字符 + 问句 >20% → "阐释型"
- 感叹号密度 >5/100 字 → "高表达欲"
- 问句 >30% → "询问导向型"

### 1.3 情绪倾向

**方法**：
- 简易情绪词典（积极/消极/中性 各 100+ 词）
- 不调 LLM，本地计算快

**注意**：
- 钉钉工作场景中"谢谢""麻烦了"是礼貌词，**不算积极情绪**
- "笑死""666""哈哈" 是娱乐词，**算积极**
- "不行""卡""挂""炸" 是负面词，**算消极**

## 二、行为模式分析

### 2.1 活跃时段热力图

**方法**：
- 把所有消息按"小时（0-23）×星期（0-6）"分桶
- 生成 7×24 矩阵
- 用 matplotlib 渲染热力图（冷色=低频，暖色=高频）

**判断标准**：
- 工作日 9-11 / 14-17 高峰 → "标准工作节奏"
- 工作日 21-23 仍有显著活动 → "下班后仍在工作"
- 周末高峰 → "无明显周末边界"
- 凌晨 0-5 活跃 → "夜猫子型"

### 2.2 协作 Top N

**方法**：
- 统计"自己 @谁" + "和谁私聊" 的频次
- 取 Top 10

**输出**：
- 协作最频繁的同事
- 协作分布（top3 占总协作量比例 → "集中型"或"广泛型"）

### 2.3 群聊 vs 单聊

**方法**：
- 按 conversationType 分桶
- 计算比例 + 平均消息长度差异

**判断标准**：
- 群聊 >80% → "群导向型"（社交/工作协调）
- 单聊 >60% → "关系导向型"（深聊一对一）
- 比例均衡 → "双轨型"

### 2.4 消息长度分布

**方法**：
- 字符数直方图
- 中位数 / 均值 / P90 / P99

**判断标准**：
- 分布右偏（少数长消息拉高均值）→ "偶尔长阐释型"
- 分布集中（多数短）→ "简洁应答型"

## 三、趣味推断（明确标 🔮）

### 3.1 职场 SBTI（基于行为）

**注意**：SBTI 不是科学框架，是娱乐标签。

**实现方式**：基于表达 DNA + 行为模式，给出 4-6 个 SBTI 标签的吻合度评分（0-100），取最高的 2-3 个。

| 类型 | 触发证据 |
|------|----------|
| **深潜者 DeepDiver** | 平均消息长度 >80 字符 + 阐释型 + 业务词 >40% |
| **甩手掌柜 HandsOff** | 群聊占主导 + 短回复比例高 + @ 别人多 |
| **火警 Firefighter** | 紧急词("卡/挂/急") 频率高 + 工作日全时段活跃 |
| **架构师 Architect** | 抽象词（"体系/系统/架构/模式"）高 + 长消息 + 提问少 |
| **侦察兵 Scout** | 问句密度 >25% + 短消息 + 群聊 + 凌晨活跃 |
| **记者 Journalist** | 收集类词（"数据/截图/记录/统计"）高 + @ 别人获取信息 |

**输出格式**：
```
🔮 职场 SBTI：架构师（吻合度 78%）
  证据：抽象词 12.3%（+4.1% vs 群体均值）、均长 87 字、问句 8%
```

### 3.2 MBTI 4 维度推断

**注意**：MBTI 信效度争议大，**只做参考**。

**实现方式**：基于 4 个维度的行为特征分别打分（0-100），取高的方向。

| 维度 | E（外向）↔ I（内向）| S（实感）↔ N（直觉）| T（思考）↔ F（情感）| J（判断）↔ P（感知）|
|------|---------------------|---------------------|---------------------|---------------------|
| 特征 | 群聊多 vs 单聊多 | 具体词 vs 抽象词 | 客观词 vs 情感词 | 计划词 vs 探索词 |
| 证据词 | E: 同步/拉群/@<br>I: 私聊/单独/直接 | S: 数据/数字/截图<br>N: 体系/模式/本质 | T: 效率/对错/逻辑<br>F: 感受/氛围/感谢 | J: 计划/截止/完成<br>P: 试试/想到/假设 |

**输出格式**：
```
🔮 MBTI：INTJ
  E/I：72 分 → I（单聊占 65%）
  S/N：63 分 → N（抽象词 14%，群体均值 8%）
  T/F：68 分 → T（情感词 3.2%，群体均值 7.8%）
  J/P：71 分 → J（计划词 6.1%，群体均值 4.2%）
```

### 3.3 最像的动物

**实现方式**：基于特征匹配，从 12 个动物原型中选最匹配的 1-2 个。

| 动物 | 特征 |
|------|------|
| **猫头鹰 Owl** | 夜间活跃 + 信息收集者 + 凌晨消息多 |
| **蜜蜂 Bee** | 高频短消息 + 协作广 + 群聊多 |
| **树懒 Sloth** | 消息少 + 周末几乎不发 + 工作日缓慢 |
| **章鱼 Octopus** | 跨多群 + 同时处理多线 + @ 别人多 |
| **狼 Wolf** | 小群核心 + 私聊多 + 协作集中 |
| **孔雀 Peacock** | 表情密度高 + 感叹号多 + 高表达欲 |
| **乌龟 Turtle** | 慢但稳 + 计划导向 + 长消息 + 文档多 |
| **狐狸 Fox** | 巧妙表达 + 短消息 + 提问多 |
| **海狸 Beaver** | 文档型 + 表格型 + 工作产出导向 |
| **海豚 Dolphin** | 团队协作 + 私聊 + 早安晚安 |
| **狮子 Lion** | 群主/管理员多 + @all 频率高 + 决策词多 |
| **变色龙 Chameleon** | 不同群风格差异大 + 适应性强 |

**输出格式**：
```
🔮 最像的动物：猫头鹰
  特征匹配：夜间活跃（22-2 点消息占 18%）、信息收集（问句 25%）、独居倾向（单聊占 60%）
  不太像：狮子（你不是 @all 型）、蜜蜂（你不是高频短消息型）
```

### 3.4 性格标签 Top 3

**实现方式**：从预设 30+ 标签池中取 Top 3。

- 「逻辑控」「细节狂」「速度型」「品质型」「团队型」「独立型」
- 「数据驱动」「直觉驱动」「体系派」「行动派」
- 「早起型」「夜猫子型」「全时型」
- 「问题终结者」「提问者」「倾听者」
- 「细节敏感」「大局观」
- 「保守型」「激进型」

## 四、与上次 diff

**实现方式**：
- 读 `data/last_snapshot.json` 的 `last_summary` 字段
- 逐项对比：
  - 关键词 Top 50 变化（新进/掉出 Top 50 的）
  - 协作 Top 10 变化
  - 活跃时段偏移
  - 情绪词密度变化
  - MBTI 维度分变化（>5 分记一次变化）

**输出格式**：
```
🔄 与上次（2026-05-12）对比：
  📈 新进 Top 关键词：deadline、专利、导师
  📉 掉出 Top 关键词：xxx、yyy
  🔁 协作变化：与@张三 协作量 +47（2x）
  ⏰ 活跃时段：晚 9-11 点活跃度 +28%
  🧠 MBTI：J/P 分从 65 → 71（更判断导向）
```

## 五、不做的事

- ❌ **不调 LLM 做心理诊断**（避免幻觉 + 法律风险）
- ❌ **不做星座/血型/属相**（信效度更低，且容易冒犯）
- ❌ **不预测未来行为**（"你下季度会..." 这种断言）
- ❌ **不评价"好坏"**（"你的工作方式很优秀" 这种无信息量的话）
- ❌ **不做"健康建议"**（工作强度分析可以，"少熬夜多休息"超出范围）
