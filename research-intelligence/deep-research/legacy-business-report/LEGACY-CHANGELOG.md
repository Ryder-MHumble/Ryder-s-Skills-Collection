## 关键设计决策（跨版本）

| 决策 | 决定 | 原因 |
|------|------|------|
| 默认输出格式 | 单一 .docx | 用户可编辑 + 可点击超链接 + 复杂表格 + 60-150KB |
| 标题 vs 文件名 | 分离 | 标题决策导向 / 文件名机器友好 |
| callout 块 | 不用 | 麦肯锡风要求克制 |
| emoj | 不用 | macOS Word 兼容性问题 |
| evidence_id 位置 | 正文内联 | 读者效率最高 |
| 5W2H 反问 | 主动触发 | 避免"形式完整内容空泛"报告 |
| 色彩系统 | 2 色（深蓝 + 中性灰） | 麦肯锡风克制 |
| Banner 实现 | matplotlib | 跨平台 + 灵活 + 轻量 |
| 信源分级 | A/B/C/Social + 估算 | 学术权威性 + 商业可用性 + 社区声音 |
| 反幻觉审计 | 9 条 checklist | 7 条事实 + 2 条格式 |
